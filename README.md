# Arcade Portal | Cyberpunk Games

A premium collection of classic arcade games with a modern cyberpunk aesthetic.

## 🕹️ Games Included

- **Battle of Hands**: High-stakes Rock, Paper, Scissors.
- **Neon Strategy**: Cyberpunk Tic-Tac-Toe.
- **Neural Recall**: Sharpen your focus with Memory Match.
- **Neon Typing**: Master the keys with lightning speed.
- **Cyber-Snake**: Navigate the neon grid and grow your trail.
- **Neon Nodes**: Tap the active nodes to contain the viral outbreak.
- **Flappy Neon**: Navigate a high-voltage spark through obstacles.
- **Neon Breakout**: Clear data bricks from the system core.

## 🚀 Features

- **Cyberpunk UI**: Vibrant neon colors and glassmorphism design.
- **Profile Tracking**: Local rank tracking based on your performance.
- **Admin Portal**: Access to system settings and game management.
- **Persistent Stats**: Your progress is saved locally.

## 🛠️ Built With

- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## 🎨 Design

Designed with a focus on modern web aesthetics, featuring smooth gradients, micro-animations, and a responsive layout.
